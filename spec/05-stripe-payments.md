# [SUPERSEDED] Stripe Payments

**This document has been replaced by [05-payments.md](./05-payments.md).**

Pragati chose to use **Square + Zelle** side by side instead of Stripe, based on committee feedback in November 2026:

- Existing Square account already tied to Pragati's bank
- Zelle preferred by many members (zero fees for the org)
- No refunds handled via the website (simplifies both rails)

See [05-payments.md](./05-payments.md) for the current spec.
